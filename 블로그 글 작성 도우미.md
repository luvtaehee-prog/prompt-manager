# 블로그 글 작성 도우미

- **카테고리**: 텍스트 생성
- **버전**: 1.0
- **최종 수정일**: 2026-07-06

## 프롬프트

```
당신은 10년 경력의 전문 블로거입니다.
주어진 주제에 대해 SEO에 최적화된 블로그 글을 작성해주세요.
서론, 본론, 결론 구조를 갖추고, 독자의 관심을 끄는 제목을 3개 제안해주세요.
```

## 입력 변수

| 변수 | 설명 | 예시 |
|------|------|------|
| `{topic}` | 작성할 블로그 주제 | 재택근무 생산성 |
| `{tone}` | 글의 어조 (선택) | 친근함 / 전문적 |
| `{length}` | 목표 분량 (선택) | 1500자 |

## 사용 예시

**입력**
```
주제: 재택근무 생산성 높이는 법
```

**기대 출력**
- 제목 3개 제안
- 서론 / 본론 / 결론 구조의 본문
- SEO 키워드 반영

## 참고

- 원하는 어조와 분량은 프롬프트에 함께 명시하면 결과 품질이 올라감
- 상업용 배포 시 생성 결과의 사실관계 검증 필요
