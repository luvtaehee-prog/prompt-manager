# 교재 표지 이미지 생성 프롬프트 — 영어 과목

**사용법**: 이 문서 전체를 이미지 생성 AI(예: Midjourney, DALL·E, Nano Banana, Ideogram 등)의 입력창에 붙여넣거나, 아래 [최종 프롬프트] 블록만 복사하여 사용한다. 변수(제목, 레벨, 대상)는 사용 전 교체한다.

---

## 1. 목표

영어 과목 교재의 **앞표지 이미지** 1종을 생성한다. 인쇄용 표지 배경 및 타이틀 레이아웃을 포함한다.

## 2. 편집 변수 (사용 전 교체)

| 변수 | 예시 값 | 설명 |
|------|---------|------|
| `{제목}` | 문법의 축 | 교재 메인 타이틀 |
| `{부제}` | English Grammar KEY | 영문 부제 |
| `{레벨}` | Green Level | 레벨/시리즈 표기 |
| `{대상}` | 중·고등 내신 대비 | 학습 대상 |
| `{출판}` | AXISOLVE | 브랜드/출판명 |

## 3. 디자인 규격

### 3-1. 판형·비율
- 비율: 세로형 (A4 기준, 3:4 또는 210×297mm)
- 해상도: 인쇄 대응 고해상도 (최소 300dpi 상당)
- 여백: 상하좌우 안전 여백 확보, 타이틀은 중앙~상단 배치

### 3-2. 색상
- 메인 컬러: 딥 그린 `#0E5C3F`
- 배경: 웜 페이퍼 톤 (아이보리/크림 계열)
- 포인트: 골드 또는 오프화이트 대비색
- 전체 톤: 채도 낮은 차분한 톤, 고급 문제집 느낌

### 3-3. 타이포그래피 (레이아웃 참고)
- 한글 타이틀: 고딕/명조 대비가 뚜렷한 서체 (IBM Plex Sans KR 계열)
- 영문 부제: 세리프 (Fraunces 계열)
- 레벨/대상: 소형 산세리프, 하단 배치

### 3-4. 그래픽 모티프
- 핵심 상징: "축(AXIS)" — 중심선, 좌표축, 또는 정렬된 격자 구조를 절제된 기하 요소로 표현
- 영어 과목 상징: 알파벳 세리프 글리프, 텍스트 라인, 밑줄/괄호 등 문법 기호를 미니멀하게 활용
- 과도한 일러스트·캐릭터·사진 배제. 편집디자인 중심의 정제된 인상

## 4. 스타일 규칙 (금지 사항)

1. 실제 인물 사진·얼굴 미포함
2. 저작권 캐릭터·브랜드 로고 미포함
3. 왜곡된 텍스트·의미 없는 문자 나열 최소화 (타이틀 텍스트는 정확히 반영)
4. 지나친 그라데이션·네온·3D 광택 배제
5. 잡지식 화려함이 아닌, 학습 교재의 신뢰감 있는 무게감 우선

## 5. 최종 프롬프트 (복사용)

### 5-1. 영문 프롬프트 (Midjourney / DALL·E 등)

### 5-2. 한글 프롬프트 (국문 지원 모델용)

```
영어 문법 교재 앞표지 디자인. 세로형 A4(3:4) 판형.
딥 그린(#0E5C3F)을 메인 컬러로, 웜 페이퍼(아이보리) 질감 배경 위에 골드 포인트.
중심 모티프: 절제된 기하학적 '축(AXIS)' — 수직 중심선과 정렬된 격자로 구조·정밀함 표현.
문법 기호(괄호, 밑줄, 세리프 알파벳)를 미니멀 그래픽 요소로 배치.
레이아웃: 상단 중앙에 굵은 한글 타이틀, 아래 세리프 영문 부제, 하단에 소형 산세리프 레벨·대상 표기.
인물·사진·캐릭터·로고 없음. 채도 낮고 차분하며 신뢰감 있는 고급 교재 톤.
편집디자인 중심, 인쇄용 고해상도(300dpi).

타이틀: "{제목}"
부제: "{부제}"
레벨: "{레벨}" / 대상: "{대상}" / 출판: "{출판}"
```

## 6. 변형 옵션 (선택)

- **표지 색상 변형**: 레벨별로 그린→블루→버건디로 컬러만 교체하여 시리즈화
- **모티프 강도**: '축' 그래픽을 강조하려면 프롬프트에 `bold central axis graphic, larger` 추가
- **텍스트 정확도**: 타이틀 텍스트 왜곡 시, 이미지 생성 후 별도 편집 툴에서 실제 폰트로 타이틀 재입력 권장
