## 5. 프로그램 기능 목록

| 번호 | 기능 | 설명 |
|---|---|---|
| 1 | 프롬프트 추가 | 제목·내용·카테고리 입력하여 등록 |
| 2 | 프롬프트 목록 | 전체 프롬프트를 번호와 함께 출력 |
| 3 | 카테고리별 조회 | 선택 카테고리 프롬프트만 출력 |
| 4 | 프롬프트 검색 | 제목·내용 키워드 검색 |
| 5 | 프롬프트 상세 보기 | 번호 입력 시 전체 내용 출력 |
| 6 | 즐겨찾기 관리 | 즐겨찾기 추가/해제 |
| 7 | 즐겨찾기 목록 | 즐겨찾기 프롬프트만 출력 |
| 0 | 종료 | 프로그램 종료 |

카테고리: 텍스트 생성, 이미지 생성, 영상 생성, 페르소나, 자동화, 기타
