# 제품 썸네일 생성

- **카테고리**: 이미지 생성
- **버전**: 1.0
- **최종 수정일**: 2026-07-06

## 프롬프트

```
다음 제품의 매력적인 썸네일 이미지를 생성해주세요.
밝은 배경, 제품 중심 구도, 고급스러운 조명을 적용합니다.
```

## 입력 변수

| 변수 | 설명 | 예시 |
|------|------|------|
| `{product}` | 대상 제품 | 무선 이어폰 |
| `{background}` | 배경 스타일 (선택) | 밝은 흰색 / 파스텔 |
| `{ratio}` | 이미지 비율 (선택) | 1:1 / 16:9 |

## 사용 예시

**입력**
```
제품: 무선 이어폰
배경: 밝은 흰색
비율: 1:1
```

**기대 출력**
- 제품 중심 구도의 썸네일 이미지
- 밝은 배경, 고급 조명 반영
- 지정 비율 적용

## 참고

- 제품명, 배경, 비율을 프롬프트에 명시하면 결과 일관성이 높아짐
- 상업용 사용 시 생성 이미지의 상표·저작권 요소 확인 필요
