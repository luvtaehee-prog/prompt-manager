# 뉴스 요약 프롬프트

- **카테고리**: 자동화
- **버전**: 1.0
- **최종 수정일**: 2026-07-06

## 프롬프트

```
다음 뉴스 기사를 3문장으로 요약해주세요.
핵심 사실만 포함하고 주관적 해석은 제외합니다.
```

## 입력 변수

| 변수 | 설명 | 예시 |
|------|------|------|
| `{article}` | 요약할 뉴스 기사 원문 | 기사 전문 텍스트 |
| `{sentences}` | 요약 문장 수 (선택) | 3 |
| `{language}` | 출력 언어 (선택) | 한국어 |

## 사용 예시

**입력**
```
기사: (뉴스 기사 원문)
문장 수: 3
```

**기대 출력**
- 3문장 요약
- 핵심 사실만 포함
- 주관적 해석 제외

## 참고

- 문장 수와 출력 언어를 명시하면 결과 형식이 고정됨
- 자동화 파이프라인 연동 시 입력 토큰 길이 제한 확인 필요
