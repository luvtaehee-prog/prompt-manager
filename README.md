# 나만의 프롬프트 관리 프로그램

터미널에서 실행하는 콘솔 기반 프롬프트 관리 프로그램입니다. 프롬프트를 카테고리별로 분류하고 검색하며 즐겨찾기로 관리할 수 있습니다.

## 실행 방법

python3 main.py

Python 3.10 이상 필요.

## 기능 목록

1. 프롬프트 추가 - 제목, 내용, 카테고리를 입력하여 새 프롬프트 등록
2. 프롬프트 목록 - 등록된 모든 프롬프트를 번호와 함께 출력
3. 카테고리별 조회 - 선택한 카테고리의 프롬프트만 출력
4. 프롬프트 검색 - 제목 또는 내용에 키워드가 포함된 프롬프트 검색
5. 프롬프트 상세 보기 - 번호를 입력해 프롬프트 전체 내용 확인
6. 즐겨찾기 관리 - 프롬프트의 즐겨찾기 추가/해제
7. 즐겨찾기 목록 - 즐겨찾기된 프롬프트만 모아서 출력
0. 종료

## 카테고리

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타

## 데이터

프롬프트는 리스트와 딕셔너리로 저장되며, 각 프롬프트는 제목, 내용, 카테고리, 즐겨찾기 여부를 포함합니다. 프로그램 실행 중 추가한 데이터는 종료 시 초기화됩니다.
